S목 (S-Mok)

엘리베이터 설치 현장용 단일 화면 Android 계산기. 두 다림추 사이 가로(A)·세로(B)를 입력하면 대각선(C)과 센터(C/2)를 즉시 표시합니다. 햇빛 아래에서도 한눈에 보이도록 흰 바탕 + 두꺼운 검은 테두리 + 형광 노랑/초록 강조의 브루탈리즘 디자인을 채택했습니다.

주요 기능

  • 실시간 대각선·센터 계산 (mm, 소수 첫째 자리)
  • 규격 프리셋 (1100×1100 / 900×900 / 700×700) + 사용자 정의 추가·편집·삭제
  • 볼륨 키로 프리셋 전환 (시스템 볼륨 바 차단)
  • JSON 내보내기·가져오기·공유 (Storage Access Framework + FileProvider)
  • 광고 없음 / 인터넷 권한 없음 / 외부 데이터 전송 없음
